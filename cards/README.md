# Cards

The **Cards** section lets you manage all your Stable Mesh virtual Mastercards — view balances, control spending, and monitor activity.

![Cards List](.gitbook/assets/02_cards_list.png)

---

## Cards List

Click **Cards** in the left sidebar to see all your virtual cards. The table shows:

| Column | Description |
|--------|-------------|
| **Alias** | The name you gave the card |
| **Card Number** | Last 4 digits of the card |
| **Available Balance** | Spendable USD currently on the card |
| **Status** | `ACTIVE` (green) or `FROZEN` (orange) |

---

## Card Detail Panel

Click any card row to open the detail panel on the right side.

![Card Detail](.gitbook/assets/04_card_detail.png)

The panel shows:

- **Card image** — masked card number and network (Mastercard)
- **Action buttons** — View Info, Freeze/Unfreeze, Delete
- **Available balance** — with a Recharge button to top up
- **Transaction history** — full spending log scrollable below

---

## Card Actions

| Action | Description |
|--------|-------------|
| [View Sensitive Info](sensitive-info.md) | Reveal full card number, expiry, and CVV |
| [Freeze / Unfreeze](freeze-unfreeze.md) | Instantly block or resume spending |
| [Top Up](top-up.md) | Load funds from your crypto wallet |
| [View Transactions](transactions.md) | Browse the card's spending history |
