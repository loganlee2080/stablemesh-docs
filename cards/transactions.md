# View Transactions

Every card maintains a complete log of its spending activity. Use this to track purchases, spot issues, and reconcile expenses.

---

## How to View Transactions

1. Go to **Cards** and click any card.
2. In the detail panel, scroll down to the **消费记录** (Transaction History) section.
3. Transactions are listed in reverse chronological order (newest first).

---

## Transaction Details

Each transaction entry shows:

| Field | Description |
|-------|-------------|
| **Merchant** | Name of the store or service charged |
| **Date & Time** | When the transaction occurred |
| **Amount** | Value in USD (negative = charge, 0 = authorisation only) |
| **Status** | Outcome of the transaction |

---

## Transaction Statuses

| Status | Meaning |
|--------|---------|
| `COMPLETED` | Transaction was successful and funds deducted |
| `FAILED` | Transaction was declined or failed |
| `REVERTED` | Transaction was reversed and funds returned |

---

{% hint style="info" %}
If you see an unexpected transaction, freeze the card immediately using the **snowflake icon** in the detail panel, then contact Stable Mesh support.
{% endhint %}
