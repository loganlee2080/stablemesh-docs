# 邀请赚钱

将 Stable Mesh 推荐给朋友，每当有人通过您的邀请链接注册并使用平台，您们双方都能获得奖励。

---

## 操作步骤

1. 点击顶部导航栏的**邀请赚钱**。
2. 复制您的专属邀请链接。
3. 分享给朋友、同事或您的社群。
4. 对方注册并使用 Stable Mesh 后，**双方均可获得奖励**。

---

## 推广建议

- 在加密社群和 Telegram 群组中分享
- 在博客文章或社交媒体中附上您的链接
- 如果您将 Stable Mesh 用于商业报销，可推荐给您的团队

---

{% hint style="info" %}
奖励条款和返佣比例请在应用内**邀请赚钱**页面查看，该页面会显示最新的奖励计算和发放方式。
{% endhint %}
