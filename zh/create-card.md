# 创建卡片

Stable Mesh 支持申请多张虚拟 Mastercard，每张卡片拥有独立的余额、消费控制和交易记录。

![创建卡片](../.gitbook/assets/03_create_card_form.png)

---

## 创建步骤

1. 进入左侧导航栏的**卡片**页面。
2. 点击右上角的**创建卡片**按钮。
3. 进入**开始申请卡片**界面。
4. 选择卡片类型——当前可选：

### Revo 卡

| 功能 | 详情 |
|------|------|
| **网络** | Mastercard |
| **适用范围** | 全球通用——支持线上和线下消费 |
| **费率** | 业内最低费率 |
| **类型** | 标准虚拟卡 |

5. 点击**下一步**，按照页面提示完成申请流程。

---

## 创建成功后

新卡片将出现在**卡片**列表中，初始余额为 **$0**。

开始使用前，请：

1. 从加密钱包[为卡片充值](cards/top-up.md)。
2. [查看卡片详情](cards/sensitive-info.md)以获取卡号、有效期和 CVV。

---

{% hint style="info" %}
您可以为不同用途创建多张卡片——例如一张用于订阅服务，一张用于广告投放，一张供团队使用。
{% endhint %}
