# 欢迎使用 Stable Mesh

Stable Mesh 是一个**虚拟信用卡（VCC）平台**，将您的加密资产与现实消费无缝连接。使用链上稳定币（USDT、USDC）为 Mastercard 虚拟卡充值，在全球任何支持 Mastercard 的商户消费——线上线下均可。

## 我们服务的客群

Stable Mesh 为所有需要可靠加密支付解决方案的用户而生：

| 客群 | Stable Mesh 如何帮助 |
|------|----------------------|
| **海外企业** | 向国际供应商、SaaS 工具和平台付款，无需银行繁琐流程，无跨境手续费 |
| **广告投放机构** | 在 Google、Meta、TikTok 等平台投放广告，为不同客户和项目创建独立卡片 |
| **Web3 用户** | 将链上 USDT/USDC 转化为可消费的 Mastercard 购买力，无需银行账户 |
| **游戏行业企业用户** | 订阅 Steam、在各大平台投放游戏广告，统一管理工作室支付需求 |

## 为什么选择 Stable Mesh

- **价格透明，无隐藏费用** — 统一费率，充值、兑换和消费均无额外收费
- **广泛接受，稳定可靠** — 依托 Mastercard 全球网络，卡片使用稳定流畅
- **加密快速充值** — 存入 USDT 或 USDC，几分钟内即可消费
- **最高 2% 返佣** — 通过推广和联盟计划赚取消费返现；[了解详情](affiliate.md)

## 平台功能

| 功能 | 说明 |
|------|------|
| **虚拟卡** | 创建并管理多张 Mastercard 虚拟卡 |
| **加密充值** | 使用 USDT 或 USDC 即时为卡片充值 |
| **全球消费** | 在所有支持 Mastercard 的场所使用 |
| **卡片管理** | 实时冻结、解冻和监控卡片 |
| **交易记录** | 每张卡的完整消费明细 |
| **邀请赚钱** | 推荐好友并获取奖励 |
