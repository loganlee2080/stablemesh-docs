# 查看敏感信息

出于安全考虑，完整卡号、有效期和 CVV 默认处于隐藏状态。需要消费时可随时查看。

---

## 查看卡片详情的步骤

1. 进入**卡片**页面，点击您要查看的卡片。
2. 在详情面板中，点击**眼睛图标**（查看详情）。
3. 面板将显示：
   - **完整卡号**
   - **有效期**
   - **CVV**
4. 如需重新隐藏，再次点击眼睛图标（隐藏详情）。

---

## 在线消费

获取卡片详情后，您可以：

- 在任意在线结账页面输入卡片信息
- 将卡片添加到 **Apple Pay** 或 **Google Pay**
- 在任何支持 Mastercard 的服务中使用

---

{% hint style="danger" %}
**请妥善保管您的 CVV。** 切勿向任何人泄露完整卡号、有效期或 CVV。Stable Mesh 客服人员绝不会索取这些信息。
{% endhint %}
