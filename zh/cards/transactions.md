# 查看交易记录

每张卡片都保存完整的消费记录。您可以通过交易记录追踪消费、发现异常并对账。

---

## 查看步骤

1. 进入**卡片**页面，点击任意卡片。
2. 在详情面板中，向下滚动至**消费记录**部分。
3. 交易按时间倒序排列（最新的在最上方）。

---

## 交易详情

每条交易记录包含以下信息：

| 字段 | 说明 |
|------|------|
| **商户** | 消费的商店或服务名称 |
| **日期与时间** | 交易发生的时间 |
| **金额** | 以美元计（负数表示扣款，0 表示仅授权） |
| **状态** | 交易结果 |

---

## 交易状态说明

| 状态 | 含义 |
|------|------|
| `COMPLETED` | 交易成功，资金已扣除 |
| `FAILED` | 交易被拒或失败 |
| `REVERTED` | 交易已撤销，资金已退回 |

---

{% hint style="info" %}
如发现异常交易，请立即点击详情面板中的**雪花图标**冻结卡片，并联系 Stable Mesh 客服。
{% endhint %}
