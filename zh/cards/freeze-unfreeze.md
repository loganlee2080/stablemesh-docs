# 冻结与解冻卡片

冻结卡片将**立即阻止所有新交易**，但不会删除卡片或影响余额。您可以随时解冻。

---

## 冻结卡片

如果您怀疑卡片被盗用、卡片信息泄露，或只是想暂停消费，请使用此功能。

1. 进入**卡片**页面，点击您要冻结的卡片。
2. 在详情面板中，点击**雪花图标**（冻结）。
3. 弹出确认对话框：**"确认冻结卡片？"**
4. 点击**确定**完成操作。

卡片列表中的状态将变为 **FROZEN**（橙色圆点）。

{% hint style="info" %}
冻结操作即时生效且可逆。您的余额不受影响。
{% endhint %}

---

## 解冻卡片

1. 进入**卡片**页面，点击显示 **FROZEN** 状态的卡片。
2. 在详情面板中，雪花图标变为**绿色**并标注**解冻**。
3. 点击**解冻**。
4. 确认对话框：**"确认解冻卡片？"** → 点击**确定**。

卡片状态恢复为 **ACTIVE**（绿色圆点），可正常接受交易。

---

## 状态说明

| 状态 | 含义 |
|------|------|
| 🟢 `ACTIVE` | 卡片正常，可接受交易 |
| 🟠 `FROZEN` | 卡片已冻结——不允许新交易 |
